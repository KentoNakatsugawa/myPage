# NOREL WITH

**"All in One Place" - シームレスなカーサブスクリプション体験**

NOREL WITHは、カーサブスクリプションの契約から納車、車両管理までをワンストップで完結できるモバイルファーストのWebアプリケーションです。

## コンセプト

- **The Compass**: 現在地と次のアクションを明確に示すダッシュボードUI
- **Gamification**: 車が目的地に向かって走るマップで進捗を可視化
- **Smart Asset Management**: AIが車両の市場価値をモニタリングし、乗り換えを提案
- **Centralized Admin**: 契約書類や個人情報の更新を一元管理

## 主な機能

### 認証
- LINE SSO モックフロー（LINEブランドカラーのログインボタン）

### ダッシュボード

#### NORELスコアメーター
- 車の速度メーター風デザインでスコアを表示
- ブロンズ / シルバー / ゴールド / プラチナのランク表示
- スコアアップへの感謝メッセージ

#### WITH Path（進捗ロードマップ）
6つのステーションで契約〜納車までの進捗を可視化：
1. **申込** - 前受金のお支払い
2. **契約** - 電子契約・口座登録
3. **書類** - 車庫証明・住民票の提出
4. **車両準備** - 名義変更・車検・クリーニング・整備・輸送手配
5. **納車** - 納車日の確定
6. **利用中** - AI査定・乗り換え提案

#### ミッションカード
現在のステップに応じたタスクカードを表示。ボタンクリックで次のステップへ進行（Optimistic UI）。

#### AI査定機能
- 現在の車両の推定評価額を表示
- おすすめ車両TOP3をAI分析に基づいて提案
- マッチ度、推しポイント、スペック情報

### サイドメニュー（設定・契約情報）

#### ドキュメント管理
- 領収書一覧（ダウンロード対応）
- 請求書一覧（ダウンロード対応）
- 契約書PDF表示

#### お客様情報の変更
- 住所変更
- 電話番号変更
- 勤務先情報
- 引き落とし口座

#### 証明書の更新・アップロード
- 免許証（表面・裏面）
- 保険証券
- 車検・自賠責

### その他
- **支払いウィジェット**: 次回お支払い日と金額を表示
- **AIコンシェルジュ**: フローティングチャットボット
- **ログアウト機能**

## セットアップ

### 必要条件
- Node.js 18以上
- npm

### インストール

```bash
# リポジトリをクローン
git clone git@github.com:KentoNakatsugawa/myPage.git
cd myPage

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

### ビルド

```bash
npm run build
npm run start
```

## ワークフロー（8ステップ）

| Step | フェーズ | タイトル | 説明 |
|------|---------|---------|------|
| 01 | 申込 | 前受金のお支払い (1/2) | 最初の決済 |
| 02 | 申込 | 前受金のお支払い (2/2) | 残金の支払い |
| 03 | 契約 | 電子契約書の締結 | 電子サイン |
| 04 | 契約 | 引き落とし口座の登録 | 口座設定 |
| 05 | 書類 | 必要書類の提出 | 車庫証明・住民票 |
| 06 | 車両準備 | 車両準備中 | 名義変更・整備等 |
| 07 | 納車 | 納車日の確定 | 日時・場所選択 |
| 08 | 利用中 | AI査定: 乗り換えチャンス | 市場価格・レコメンド |

## デモ操作

1. 「LINEで始める」をクリックしてログイン
2. 各ミッションカードのボタンをクリックしてステップを進める
3. ステップ3（契約）とステップ7（納車）完了時に紙吹雪エフェクト
4. ステップ8で「査定詳細・カタログを見る」をクリックして車両レコメンドを確認
5. 右上のハンバーガーメニューから設定・契約情報にアクセス
